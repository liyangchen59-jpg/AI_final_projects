# Project3 README

This project is based on the Transformer text classification implementation in `TransformerALL` and contains two experiment settings:

- **Task A**: train and test on the original text
- **Task B**: preprocess the text first (remove punctuation and compress whitespace), then train and test

## 2. Environment Setup

Python 3.10+ is recommended.

### 2.1 Install Dependencies

Run the following command in the project root directory:

```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple --default-timeout=200 
```

The `requirements.txt` file includes:

- numpy
- scikit-learn
- tqdm
- torch

If PyTorch is not installed in your environment, you may need to install the proper version separately based on your CPU/CUDA setup.

## 3. How to Run Task A and Task B

All commands below should be executed inside the `TransformerALL` directory.

### 3.1 Go to the Working Directory

```bash
cd TransformerALL
```

### 3.2 Run Task A

Task A uses the original text without extra preprocessing.

```bash
TASK_NAME=A python run.py
```

#### What Task A Does

- Loads `THUCNews/data/train.txt`, `dev.txt`, and `test.txt`
- Trains the model for 6 epochs
- Prints the following test outputs:
  - Test Loss
  - Test Accuracy
  - Precision / Recall / F1-score
  - Confusion Matrix
- Saves misclassified samples to `THUCNews/data/taskA.misclassified`
- Predicts 5 example texts at the end

#### Reference Output for Task A

According to `experiment_results/taskA_training_log.txt`:

- **Test Acc: 83.96%**
- Misclassified samples are saved to `THUCNews/data/taskA.misclassified`

Example log snippet:

```text
Running Task A
Loading data...
...
Test Loss:  0.55,  Test Acc: 83.96%
Misclassified samples saved to: THUCNews/data/taskA.misclassified
```

### 3.3 Run Task B

Task B applies text preprocessing before training:

- removes Chinese and English punctuation
- compresses extra whitespace

Run:

```bash
TASK_NAME=B python run.py
```

#### What Task B Does

- Builds the vocabulary from preprocessed text
- Trains the model for 6 epochs
- Prints test metrics and the confusion matrix
- Saves misclassified samples to `THUCNews/data/taskB.misclassified`
- Predicts 5 example texts at the end

#### Reference Output for Task B

According to `experiment_results/taskB_training_log.txt`:

- **Test Acc: 84.20%**
- Misclassified samples are saved to `THUCNews/data/taskB.misclassified`

Example log snippet:

```text
Running Task B
Loading data...
...
Test Loss:  0.52,  Test Acc: 84.20%
Misclassified samples saved to: THUCNews/data/taskB.misclassified
```

## 4. Reproducibility Steps

To reproduce the main results, run the following commands in order:

```bash
pip install -r requirements.txt
cd TransformerALL
TASK_NAME=A python run.py
TASK_NAME=B python run.py
python compare_misclassified.py
```

Explanation:

- The first two runs reproduce the training and testing results of Task A and Task B
- `python compare_misclassified.py` compares the differences between the misclassified samples of the two tasks

## 5. Compare Misclassified Samples

Run this inside the `TransformerALL` directory:

```bash
python compare_misclassified.py
```

This script reads:

- `THUCNews/data/taskA.misclassified`
- `THUCNews/data/taskB.misclassified`

and prints:

- samples misclassified in Task A but correctly classified in Task B
- samples misclassified in Task B but correctly classified in Task A

You can also refer to:

- `experiment_results/compare_result.txt`

## 7. Notes

- `run.py` switches tasks using the environment variable `TASK_NAME`:
  - `A`: no preprocessing
  - `B`: enable preprocessing
- If `TASK_NAME` is not set, the program runs **Task A** by default.
- The main model configuration is in `TransformerALL/models/Transformer.py`, including:
  - number of epochs: 6
  - batch size: 128
  - pad size: 32
  - learning rate: 5e-4
